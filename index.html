Example of my work:

Star Wars Quiz

GitHub Repo Link

I chose this project for several reasons:

  1. Its extremely thorough readme
  2. Simple but effective
  3. Great responsiveness

I planned this project out to practice JavaScript specifically. A lot of detail went into the ReadMe so that it could function as a sort of road map of things to do once I started coding. From wireframes to colour palettes and some User Stories to follow, along with deployment and forking guidelines, it had everything.


The quiz was set up with an array of questions and answers with an attribute telling if the answer was the correct one or not. I tracked the user's score in the variable currentScore which incremented by 1 if the user clicked on the button which was associated with the correct answer. I had used 3 event listeners: for showing and closing the quiz info (which opened/closed a modal box) and the all important one which started the quiz off.


The main showQuizQuestions function tracked both the user's current score and the index number of the quiz they were on. The current question object stored that index and would increment any time one of the answers was clicked, thus tracking the progress. I had to ensure the event was only fired when clicking on the buttons and not the wrapper div which I solved by ensuring the className on the clicked element was "buttons". Upon reaching the end of the quiz, the currentQuestionObject (that was tracking the question number) returns as undefined, so I used that to reset the quizIndex and currentScore variables.
